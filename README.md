# Typing Speed Test App

A simple **desktop application** built with **Python** and **Tkinter** that allows users to test their typing speed and accuracy. This project demonstrates GUI development, event handling, and basic timing/logic in Python.

---

## Features

- Random sentence generator for typing practice  
- Timer to measure typing duration  
- Calculates **Words Per Minute (WPM)**  
- Calculates **accuracy percentage**  
- Simple and clean graphical interface
